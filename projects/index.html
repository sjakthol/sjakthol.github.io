aws-s3-basic-auth

Protect access to S3 bucket with HTTP Basic Authentication. Powered by AWS CloudFront and Lambda@Edge.

aws-account-infra

CloudFormation templates for setting up an infrastructure for an AWS account. This includes configurations for CloudTrail, IAM, KMS and VPC.

aws-kms-crypt

A cross-language library for encrypting and decrypting secrets with the AWS KMS service. Includes support for Bash, Node.js and Python.
aws kms bash python nodejs

cfn-monitor

A tool for monitoring the progress of AWS CloudFormation stack events during the creation or update of a stack.
