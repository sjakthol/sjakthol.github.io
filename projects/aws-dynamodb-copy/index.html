aws-dynamodb-copy

2018-08-31
#aws  #dynamodb  #nodejs 

aws-dynamodb-copy is a simple script that allows you to copy the contents of a DynamoDB table into another table on the same AWS region.

The script doesn’t provide any fancy features. It scans the source table using a parallel scan and writes the items to the target tables in batches. The rate at which items are copied can be tuned to suit the capacity allocated to the target and source tables.

The script is able to provide a sustained rate of at least 10000 items copied per second (running on an m5.large instance). Higher rates have not been tested.
