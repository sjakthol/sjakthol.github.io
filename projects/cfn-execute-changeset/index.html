cfn-execute-change-set

A tool for reviewing and executing AWS CloudFormation change sets.

cfn-execute-change-set is a command line utility for reviewing and executing AWS CloudFormation change sets. The tool parses the output of AWS CLI create-change-set command, prints out a summary of changes and executes the change set if so instructed.

This utility requires Node.js and can be installed with npm:

npm install -g cfn-execute-change-set
