digitransit-stoptimes-android

This Android application can be used to view real-time departure information of public transportation in the Helsinki metropolitan area. The key features of this Android application are:

  • Search stops by their name
  • View stops near your current location
  • View upcoming departures from a stop with real-time predictions
  • Save favorite stops for easy access

The data is provided by HSL through the Digitransit platform.
