aws-kms-crypt

A cross-language library for encrypting and decrypting secrets with the AWS KMS service. Includes support for Bash, Node.js and Python.

aws-kms-crypt is a cross-language utility for securing the confidentiality of secrets and sensitive data in AWS. The key features of this tool are:

  • Simple APIs for encrypting and decrypting sensitive data
  • Interoperable implementations for multiple languages (Bash, Node and Python)
  • Envelope Encryption with AES-128-CBC and KMS generated data keys. That is, all secret data is encrypted locally and never sent to Amazon in plain text.
